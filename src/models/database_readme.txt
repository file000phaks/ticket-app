Database and Backend essentials
-------------------------------

Users
-----
- can get all users
- can get a single user by id or by email
- can get a list of users by their ids
- can update a user 
- can store a user after signup
- can get users of some particular similarity (e.g say i want to get all admins)
- can check for user existence

Tickets
-------
- can create and delete tickets
- can get and update tickets
- can get a ticket or a list of tickets by ids

Notifications
-------------
- can create and delete notifications
- can retrive notifications for a user by the userId
- can get a notification by its id
